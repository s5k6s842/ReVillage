3 Core Elements

🎫

The Passport (NFT)

入村パスポート ＆ 基礎投票権 (1票)。
市場売買可能。
あくまでコミュニティへの「入り口」です。

💰

The Economy (Tip)

感謝の対価としてのチップ。
ゲストから住民への一方通行。
Tipで権力（Honor）は買えません。

🛡️

The Trust (Honor)

貢献と信頼のスコア。
譲渡不可 (SBT)。
住民からの評価でのみ蓄積され、ガバナンス力を決定します。

Trust Ranks

Unified Scale: ランク基準は住民・会員共通ですが、獲得手段（貢献 vs 文化）が異なります。

Rank 条件 (Honor) 投票力 (係数) 特権
Newcomer 0 - 99 x 1.0 基本アクセス権のみ
Villager 100 - 499 x 1.2 議論参加、予約優先(小)
Guardian 500 - 999 x 1.5 プロジェクト起案権、限定エリア
Elder 1000+ x 2.0 優先審議権、拒否権
⚠️ The Decay Rule (減衰ルール)

Honorは「貯金」ではなく「筋肉」です。
活動がない期間が続くと、保有Honorは徐々に減少し、ランクも降格します。
(常に現在進行系の貢献者がリーダーシップを取る仕組み)
